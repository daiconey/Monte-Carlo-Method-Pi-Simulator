なぜこれで求まるの？

一辺 r の正方形の面積は r2。その中の扇形（半径 r、中心角 90°）の面積は 1/4 × πr2 です。
ランダムに点を打ったとき、扇形の中に入る確率は面積比と同じ π/4 になります。
よって、4 × (中に入った数 ÷ 全体の数) で π が計算できます。

学習用 Python コード 

import random
import math

def estimate_pi(num_samples):
    inside_points = 0
    for _ in range(num_samples):
        x = random.random()
        y = random.random()
        if x**2 + y**2 <= 1.0:
            inside_points += 1
    pi_estimate = 4 * inside_points / num_samples
    error = abs(pi_estimate - math.pi) / math.pi * 100
    return pi_estimate, error

def main():
    print("--- モンテカルロ法による円周率の推定 ---")
    trials = [100, 1000, 10000, 100000, 1000000]
    for n in trials:
        pi_val, err = estimate_pi(n)
        print(f"回数: {n:10,d} | 推定値: {pi_val:10.6f} | 誤差: {err:10.4f}%")

if __name__ == '__main__':
    main()
